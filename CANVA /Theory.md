1. What is Difference Between UI and UX 

- UI : user Interface
1. UI refers to the look and visual design of an app or website.
includes: Colors, Buttons, Fonts, images, layouts, and many more things
"UI define how the website or App looks " 

- UX : User Experience 
1. UX refers to how easy and comfortable the app is to use for the user. 
includes : Easy Navigations , fewer steps to complete the task, clear Instructions, smooth and fast experience, and many more things

"UX define how website or Application works and feels"

2. what is a good design and what is the bad design 
- Good Design : clear text, proper spacing, easy navigation, simple colors and decent theme or color combination, etc..

Example: whatsApp, gPay, etc..

- Bad Design : More Color, small text, confusing buttons or navigation, too many steps for one task, etc...

Ex: Government website or applications

3. What is Canva 
- Canva is an online graphic design platform that helps people create visual content easily

use of Canva 
1. Social media posts (Instagram, Facebook, LinkedIn)
2. Posters, flyers, and banners
3. Presentations and resumes
4. Logos and business cards
5. Videos and simple animations

Key Features of Canva 
1. Drag-and-drop editor
2. thousands of ready made templates 
3. free fonts, images, icons, and elements
4. easy download, and share options are available in Free 
5. team member can access 

4. What is Figma : 
- Figma is a web-based design and prototyping tool used to create 

1. UI/UX designs for websites and mobile apps 
2. Wireframes and mockups
3. Interactive prototypes
4. Design systems and components 

It allows multiple people to design together in real time. 

-- Key Features of Figma 
1. Works directly in the browser
2. Real-time collaboration with team members
3. Easy drag-and-drop interface
4. Create interactive prototypes without coding
5. Supports plugins for icons, images, and automation
6. Useful for developers to inspect design, colors, and spacing
