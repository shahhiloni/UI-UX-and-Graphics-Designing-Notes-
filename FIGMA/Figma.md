------------------------------- What is Figma ------------------------------------
- Figma is a cloud-based UI/UX design and prototyping tool used to create website designs, mobile app layouts, wireframes, and interactive prototypes.

- It works directly in the web browser, so designers can design, share, and collaborate in real time without installing heavy software. 


----------------------------------- importance of Figma -------------------------------
1. Real-time collaboration – Multiple designers and developers can work on the same design at the same time.
2. Cloud storage – Designs are saved online, so no risk of losing files.
3. Easy sharing – You can share a simple link for feedback or developer handoff.
4. Prototyping feature – Create clickable and interactive UI without coding.
5. Component & reusable design system – Helps maintain consistency in large projects.
6. Cross-platform – Works on Windows, Mac, and even in the browser.
7. Free plan available – Beginners can start learning without paying.

Figma makes designing, collaboration, and sharing faster and easier, which is why most modern UI/UX designers prefer it.

--------------------------------- 4 Steps to Make a website or Application in Figma -------------------
1. Explore the existing Application or Website which is related to your topic 
2. Create a Flow Chart Accordingly 
3. Create Wireframe according to Flow Chart 
4. Choose theme For Wireframe Features like background, buttons, layout, pops, alerts, and many more ....

-------- What is the difference between MockUps and WireFrames -----------
*** WireFrame : Wireframe is a basic layout of a webpage or app.
1. Structure, Placement of elements, User flow, Functionality
2. No colors, No images, No styling, Only boxes, lines, and text placeholders

EX: Where logo will be placed, Login form position, Button location

*** MockUps : A Mockup is a high-fidelity design showing how the final product will look.
It focuses on: Colors, Fonts, Images, Branding

UI appearance
- Real buttons
- Actual colors
- Typography
- Proper spacing & styling
